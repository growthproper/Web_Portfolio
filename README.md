# Web_TurtleMan
# 거북이 사회적 협동조합 홈페이지 개발
# URL : http://거북이사회적협동조합.org/index.html
## 사이트맵 구성
  * 메인 페이지
  * 공지사항
  * 조합소개
  * 사업안내
  * 활동소식
  * 후원문의
  * Contact Us
 ## 사용기술
  * bootstrap
  * css3
  * html5
  * javascript
  * jquery
  * 템플릿을 활용하여 사이트 컨셉에 맞도록 코드 일부 수정 및 레이아웃 일부 변경
  * 반응형 웹
